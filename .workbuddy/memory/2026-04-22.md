### 立定跳远距离算法 V3

**从 V1 升级到 V3，改动 2 个文件：**

1. `src/services/counters/StandingLongJumpCounter.ts` — 完全重写（445 行）
   - **身体比例标定**：自动检测用户稳定站立，用躯干长度(肩中→髋中)作为标尺
     - 标定公式：pixelsPerCm = torsoLengthPx / (userHeightCm × 0.29)
     - 躯干长度 ≈ 身高 × 29%（骨骼测量均值）
     - 默认身高 170cm，可通过 setUserHeight() 修改
   - **完整状态机**：idle → ready → crouch → takeoff → airborne → landing → stable
     - idle：等待稳定站立，收集标定数据（25帧方差 < 5）
     - ready：等待下蹲
     - crouch：膝盖角 < 100°，记录蹲深
     - takeoff：膝盖角 > 140°，记录起跳脚踝位置
     - airborne：脚踝Y上升 > 15px，追踪最高点
     - landing：脚踝Y回落，等待15帧稳定
     - stable：记录距离，1.5秒后自动回到 ready
   - **Kalman 滤波**：膝盖角度 + 脚踝XY 三路独立滤波
   - **距离计算**：水平位移(像素) / pixelsPerCm → 厘米
   - **合理性检查**：50cm ~ 400cm 范围过滤
   - **内置 getFeedback**：标定中提示 / 蹲深不足 / 起跳腾空 / 落地质量 / 成绩反馈
   - 内置 SlidingWindow + KalmanFilter1D（不依赖外部模块）

2. `src/hooks/useExerciseFeedback.ts` — 新增立定跳远专用姿态检测
   - 新增 `checkLongJumpForm()`：背部前倾检测(>45°)、膝盖深度(<80°过深)、半蹲蓄力提示(130°~160°)、双脚落地对称性检测
   - `standing_long_jump` 和 `vertical_jump` 反馈独立路由（之前合并为 checkJumpForm）

   - **合理性检查**：10cm ~ 120cm 范围过滤
   - **内置 getFeedback**：蹲深不足 / 起跳腾空 / 落地缓冲 / 成绩评价 + 最佳成绩对比

### 仰卧起坐计数器 V3（新增项目）

**改动 5 个文件：**

1. `src/types/index.ts` — ExerciseType 新增 `'sit_ups'`
2. `src/constants/exerciseConfig.ts` — 新增仰卧起坐配置（默认目标 40 次）
3. `src/services/counters/SitUpCounter.ts` — 新建（380 行）
   - **核心算法**：肩-髋-膝三点躯干角度 (trunk angle) 状态机
     - 仰卧时角度 ≈ 140°~180°（接近平躺）
     - 坐起时角度 ≈ 50°~85°（躯干前倾，肘触膝）
   - **状态机**：idle → lying → rising → up → returning → done → lying（循环）
     - idle：等待用户躺到垫上
     - lying：仰卧姿态，采集臀部基线
     - rising：正在起身（角度减小）
     - up：坐起到位（角度 ≤ 85°，保持 4 帧确认）
     - returning：返回仰卧（角度增大）
     - done：完成一次，计数+1，200ms 冷却后进入 lying
   - **中考规则适配**：
     - 坐起判定：角度 ≤ 85° + 保持 4 帧
     - 仰卧判定：角度 ≥ 140° + 保持 5 帧
     - 犯规检测：臀部离垫（hip_lift）、动作太快（too_fast）、不完整（incomplete）
     - 速度自适应：记录最近 10 次周期帧数，动态调整判定窗口
   - **Kalman 滤波**：躯干角度 + 肩部Y + 髋部Y 三路
   - **防抖机制**：最小周期 12 帧（≈0.4s），最大 90 帧（≈3s）
   - **方向检测**：通过角度历史窗口检测变化方向，防止误切换
   - **肘-膝距离验证**：坐起到位时辅助验证肘部是否触膝（用肩部代替肘部）

4. `src/hooks/useWorkout.ts` — 注册 SitUpCounter
5. `src/hooks/useExerciseFeedback.ts` — 新增 `checkSitUpForm()`
   - 膝盖角度检测（大小腿应呈 90°：>130° 太直、<50° 太紧）
   - 臀部离垫检测（髋踝距离 < 30px）
   - 起身过程鼓励反馈
   - 坐起到位反馈
   - 双脚对称性检测
